# מדריך התקנה והגדרה

עקבי אחר השלבים הבאים כדי להתקין את המערכת על המחשב המקומי שלך.

## דרישות מערכת
* **Node.js:** גרסה 18.0.0 ומעלה.
* **מנהל חבילות:** npm או yarn.
* **זיכרון:** לפחות 4GB RAM פנוי.

## שלבי ההתקנה

### 1. התקנת החבילה
יש להריץ את הפקודה הבאה בטרמינל כדי להתקין את כלי השירות:
```bash
npm install project-system --save-dev
